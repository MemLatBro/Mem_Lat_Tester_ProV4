# Mem_Lat_Tester_ProV4
Copy the Code file and save as a .py file and run it.
Intro: This is an indepth memory latency grapher for RAM over clockers as well as a mild stress test! 

This is meant to be an In depth memory latency grapher it will produce 3 files an HTML file that will open in browser

to review the implications of your results and give you a graded score, A text file with the raw numbers at each 

memory size in MB/s and NS, and a image file with the graphs only so you can see your memory heirarchy ladder.

There are technically 4 options but Mainly "Full Run" and "Quick Run" 

The Full run can take up to 25 Minutes! but its worth having all the results it is a complete memory access profile.

The Quick run takes about 12-15 minutes and your RAM latency score will be 10-12 ns faster than it actually is if full 

ladder applications like Aida 64.
